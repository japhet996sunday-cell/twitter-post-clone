# Twitter Post Clone

A responsive Twitter-style post input clone built with HTML, CSS, and JavaScript.  
It allows users to type a post, see live character count, and display the posted content on the screen.

---

## 🚀 Features

- Live character counter (0 / 200 limit)
- Warning indicator when approaching limit
- Post preview display
- Input validation (no empty posts)
- Fully responsive design (mobile friendly)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📱 Responsiveness

The layout adapts to different screen sizes:
- Desktop view: centered card layout
- Mobile view: responsive width adjustment

---

## 📸 How It Works

1. Type a message in the textarea  
2. See live character count update  
3. Click **Post** to display the message below  
4. Input resets after posting
